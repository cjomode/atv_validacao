UC-02: Realizar soma com sucesso

- Descrição: Verificar se o sistema realiza corretamente a soma de dois números positivos.
- Pré-condições: Calculadora funcional 
- Passos: 
    1. Usuário insere um número positivo;
    2. Seleciona a operação de soma;
    3. Insere outro número positivo;
    4. Usuário pressiona "="
- Dados de entrada: 5, +, 2
- Resultado esperado: O sistema exibe um valor válido e positivo (7)

UC-09 Transição de Estados: Operações encadeadas
- Descrição: Verificar se o resultado de uma operação pode ser usado na operação seguinte.
- Pré-condições: Calculadora funcional.
- Passos:
    1. Usuário insere 5 e 5, seleciona soma e obtém 10;
    2. Usa o resultado (10), multiplica por 2 e obtém 20;
    3. Usa o resultado (20), subtrai 5 e obtém 15.
- Dados de entrada: (5+5)=10 → (10*2)=20 → (20-5)=15
- Resultado esperado: O sistema exibe 15.
