UC-02: Realizar soma com sucesso

- Descrição: Verificar se o sistema realiza corretamente a soma de dois números positivos.
- Pré-condições: Calculadora funcional 
- Passos: 
    1. Usuário insere um número positivo;
    2. Seleciona a operação de soma;
    3. Insere outro número positivo;
    4. Usuário pressiona "="
- Dados de entrada: 5, +, 2
- Resultado esperado: O sistema exibe um valor válido e positivo (7)
